# ai-note-h5 优化方案

## 1. 生成过程步骤提示

**Files:** `src/pages/Home.tsx`, `src/pages/Home.scss`

**做法：** 把 `loading` 状态从 boolean 改为枚举 `idle | fetching | analyzing | done`

**效果：**
- 输入链接点生成 → "正在抓取网页..."（fetching）
- 抓取完成 → "AI 分析中..."（analyzing）
- 渲染完成 → 跳转笔记页

**改动量：** ~30 行

---

## 2. 生成失败加重试按钮

**Files:** `src/pages/Home.tsx`

**做法：** 在 error 显示区域加一个"重试"按钮，onClick 直接调用 `handleGenerate()`（保持当前输入不变）

**改动量：** ~5 行

---

## 3. 笔记卡片加文字摘要

**Files:** `src/pages/Notes.tsx`, `src/pages/Notes.scss`, `src/pages/Notes.tsx`

**做法：** 列表里的 NoteIndexItem 加上 `snippet` 字段（存到 NoteData 时取 textContent 前 50 字），Notes 列表显示摘要

**改动量：** ~15 行

---

## 4. 深色模式

**Files:** `src/App.scss`, `src/pages/Home.scss`, `src/pages/Notes.scss`, `src/pages/Settings.scss`

**做法：** 
- 所有页面已经有 CSS 变量，只需在 `:root` 加 `dark` 类覆盖变量
- 在 settings 页面加一个 toggle，存到 localStorage
- NoteIframe 也需要注入深色的 `themeOverride` CSS

**改动量：** ~40 行

---

## 5. 复制分享链接

**Files:** `src/pages/Note.tsx`

**做法：** 在 Note 页面顶栏加一个"复制链接"按钮，点击后复制 `https://ai-note-h5-1351746008.cos-website.ap-guangzhou.myqcloud.com/#/note/${id}` 到剪贴板

**改动量：** ~8 行

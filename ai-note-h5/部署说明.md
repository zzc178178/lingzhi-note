## 手动部署方式

### 腾讯云 COS

> ⚠️ 使用 COS 必须**关闭翻墙**，用国内网络直连。翻墙状态下访问会超时或拒绝连接。
> ⚠️ COS 刷新时可能触发 HTML 下载，这是平台限制，建议主要使用 Netlify。

**方式一：一键脚本（推荐）**
```bash
python deploy.py cos
```

**方式二：手动上传**
```bash
# 1. 构建
npm run build

# 2. 复制到 deploy 目录
rm -rf deploy && cp -r dist deploy

# 3. 运行 Python 上传脚本
python -c "
from qcloud_cos import CosConfig, CosS3Client
import os

config = CosConfig(SecretId='YOUR_SECRET_ID',
                   SecretKey='YOUR_SECRET_KEY',
                   Region='ap-guangzhou')
client = CosS3Client(config)
b = 'ai-note-h5-1351746008'
deploy = r'D:/AI-PM/AI_coding/ai-note-make/ai-note-h5/deploy'

for root, dirs, files in os.walk(deploy):
    for fname in files:
        local = os.path.join(root, fname)
        key = os.path.relpath(local, deploy).replace(os.sep, '/')
        ct = 'text/html; charset=utf-8' if fname.endswith('.html') else \
             'text/css' if fname.endswith('.css') else \
             'application/javascript' if fname.endswith('.js') else 'text/plain'
        with open(local, 'rb') as fh:
            client.put_object(Bucket=b, Key=key, Body=fh, ContentType=ct,
                              ContentDisposition='inline')
        print(f'  ✓ {key}')
print('COS deployed')
"
```

### Netlify（推荐，最稳定）

**方式一：手动拖拽（最简单）**
1. 运行 `npm run build` 构建项目
2. 打开 https://app.netlify.com/drop
3. 把 `ai-note-h5/deploy` 文件夹**拖拽**到网页上
4. 等待几秒，部署完成

> 注意：上传 `deploy/` 文件夹即可，不要上传整个 `ai-note-h5` 项目（虽然 Netlify 会自动构建，但直接上传构建产物更快更稳定）

**方式二：CLI 命令**
```bash
npx netlify deploy --prod --dir=deploy --allow-anonymous
```

**方式三：连接 GitHub 自动部署**
1. 打开 https://app.netlify.com/
2. 点击 "Add new site" → "Import an existing project"
3. 选择 GitHub 仓库 `zzc178178/lingzhi-note`
4. 配置构建命令：`npm run build`
5. 发布目录：`ai-note-h5/dist`
6. 保存后，每次 push 到 GitHub 会自动部署

## 常见问题

### COS 刷新时下载 HTML
这是 COS 静态网站托管的已知限制，与单页应用（SPA）路由冲突有关。
- **解决办法**：优先使用 Netlify 地址
- **缓解**：已设置所有文件 `Content-Disposition: inline`，但无法完全避免

### 字体不一致
如果 "灵知" 和 "笔记" 粗细不一致，检查：
1. 确认已重新构建：`npm run build`
2. 确认已重新部署到对应平台
3. 清除浏览器缓存后刷新

## 网页抓取代理

- 代理地址：`http://127.0.0.1:8765`
